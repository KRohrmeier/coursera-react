Basic Git Commands

git init
initializes the current folder as a git repository
this command will mark the folder as "master"; that is, this is now the master branch
git status
tells you the current status of the repository
git add
all files in current directory will be added to the staging area
git commit
will commit current state of folders (staged) to the git repository
a snapshot off the current state of all files is added to the git repository
-m (flag)
add the -m flag to git commit to add a message with the commit
git log --oneline
shows the 8 digit hash and the commit message of all commits thus far in repo
a brief log versus the more detailed log you'd get with git log
